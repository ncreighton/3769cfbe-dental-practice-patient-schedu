== Description ==

A WordPress plugin for dental offices and multi-location dental groups managing patient scheduling and treatment plan communications. Features online appointment booking with procedure-specific time slots, automated recall reminders for cleaning and follow-up visits, treatment plan acceptance tracking with financing option integration, patient satisfaction survey delivery, and HIPAA-compliant patient portal for viewing radiograph summaries and outstanding balance statements.

**Features:**

* Core functionality

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress admin > Plugins
3. Configure via Settings > Dental Practice Patient Scheduling and Treatment Plan WordPress Plugin
